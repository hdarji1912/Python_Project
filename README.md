# 🐍 Python Projects Collection

A collection of beginner to intermediate **Python projects** built while learning programming, improving problem-solving skills, and exploring practical applications of Python.

These projects focus on writing clean code, understanding core programming concepts, and building real-world applications.

---

# 📂 Projects Included

| Project | Description |
|----------|-------------|
| 💰 Expense Tracker | Track daily expenses and manage personal spending. |
| 🕒 Digital Clock | A real-time digital clock built using Python Tkinter. |
| 🤖 Personal Chat | A simple command-line chatbot developed in Python. |
| 📱 QR Code Generator | Generate QR Codes from URLs. |
| 🧮 Smart Calculator | A calculator supporting multiple mathematical operations. |
| ✅ To-Do List | Manage daily tasks with a simple CLI application. |
